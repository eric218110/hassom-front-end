# HASSOM
Projeto da cadeira de **Projeto de Sistemas de Informação**

### Descrição

> Loja de instrumentos musicais.
> Front end

### Tecnologias 

* ReactJS
* Styled Components
* React router DOM
* Redux e react-redux
* React icons
* React Spring
* TypeScript

## Instalação

``` bash
# build
$ yarn build
````  
````bash
# watch mode
$ yarn start
````  
````bash
# test
$ yarn test
````
